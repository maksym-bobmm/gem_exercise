# GemExercise

This is a gem building exercise. The gem add text to output of rails commands.  

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'gem_exercise', git: 'https://github.com/maksym-bobmm/gem_exercise.git'
```

And then execute:

    $ bundle

You can set your own text. For this you need to generate config file with:

    $ rails generate gem_exercise

This command add file named **gem.exercise.rb** to your **config/initialize** folder
After that you can set your own text like this:
```ruby
GemExercise::Message.configure do |config|
  config.message = 'hello world'
end
```
